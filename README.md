## Clonar el repositorio

- En la terminal: git clone https://github.com/XxemicoxX/Proyectate-FrontEnd.git

## Tener en cuenta

- Tener instalado angular: https://angular.dev/installation

- Se usa el puerto 4200

## Instalar dependencias

- Dentro de la carpeta, en la terminal:
    npm install o npm i

## Ejecucion 

- En la terminal: ng serve
